= Trabajo anulado debido a cierre
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`Job aborted due to shutdown`

* *Causa*
+
El sistema de almacenamiento se está reiniciando.

* *Acción Correctiva*
+
Vuelva a intentar el trabajo después de que se reinicie el sistema de almacenamiento.
