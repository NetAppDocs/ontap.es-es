= Volumen sin conexión
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`Volume offline`

* *Causa*
+
El volumen del cual se realiza el backup está sin conexión.

* *Acción Correctiva*
+
Coloque el volumen en línea y vuelva a intentar el backup.
