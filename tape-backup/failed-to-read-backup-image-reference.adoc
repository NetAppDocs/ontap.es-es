= Error al leer la imagen de la copia de seguridad
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`Failed to read backup image`

* *Causa*
+
La cinta está dañada.

* *Acción Correctiva*
+
Si la cinta está dañada, no podrá realizar la operación de restauración.
