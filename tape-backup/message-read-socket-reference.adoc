:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`Message from Read Socket: error_string`

* *Causa*
+

* *Acción Correctiva*
+
Compruebe la conexión de red al mando a distancia.
