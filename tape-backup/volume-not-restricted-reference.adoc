= Volumen no restringido
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`Volume not restricted`

* *Causa*
+
No está restringido el volumen de destino al que se restauran los datos.

* *Acción Correctiva*
+
Restrinja el volumen y vuelva a intentar la operación de restauración.
