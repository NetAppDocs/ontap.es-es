= Error al escribir en la cinta
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Mensaje*
+
`Tape write failed`

* *Causa*
+
La cinta utilizada para la copia de seguridad está dañada.

* *Acción Correctiva*
+
Sustituya la cinta y vuelva a intentar la tarea de copia de seguridad.
