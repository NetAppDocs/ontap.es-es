= La ruta de volumen no es válida
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Mensaje*
+
`Invalid volume path`

* *Causa*
+
No se encuentra el volumen especificado para la operación de backup o restauración.

* *Acción Correctiva*
+
Vuelva a intentar el trabajo con una ruta de volumen y un nombre de volumen válidos.
