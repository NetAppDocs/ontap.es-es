= Error al eliminar la snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`Failed to delete snapshot`

* *Causa*
+
No se ha podido eliminar la instantánea automática porque está en uso en otras operaciones.

* *Acción Correctiva*
+
Utilice `snap` el comando para determinar el estado de la snapshot. Si la instantánea no es necesaria, elimínela manualmente.
