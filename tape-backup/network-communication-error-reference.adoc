= Error de comunicación de red
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Mensaje*
+
`Network communication error`

* *Causa*
+
Se produjo un error en la comunicación a una cinta remota en una conexión triple NDMP.

* *Acción Correctiva*
+
Compruebe la conexión de red al mando a distancia.
