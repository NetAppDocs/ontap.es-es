= No se pudo cargar la nueva cinta
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`Failed to load new tape`

* *Causa*
+
Error en unidad de cinta o soporte.

* *Acción Correctiva*
+
Sustituya la cinta y vuelva a intentar la operación.
