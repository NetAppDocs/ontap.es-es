= No se crearon archivos
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Mensaje*
+
`No files were created`

* *Causa*
+
Se intentó un DAR de directorio sin permitir la funcionalidad DAR mejorada.

* *Acción Correctiva*
+
Active la funcionalidad DAR mejorada y vuelva a intentar DAR.
