= No se puede crear la snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Mensaje*
+
`Failed to create snapshot`

* *Causa*
+
El volumen ya contiene el número máximo de copias snapshot.

* *Acción Correctiva*
+
Elimine algunas copias de Snapshot y vuelva a intentar la operación de backup.
