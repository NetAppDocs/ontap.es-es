= Ndmpd session_ID no activo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`ndmpd session session_ID not active`

* *Causa*
+
Es posible que la sesión NDMP no exista.

* *Acción Correctiva*
+
Utilice `ndmpd status` el comando para ver las sesiones de NDMP activas.
