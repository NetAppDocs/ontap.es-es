= Trozos fuera de servicio
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Mensaje*
+
`Chunks out of order`

* *Causa*
+
Las cintas de copia de seguridad no se restauran en el orden correcto.

* *Acción Correctiva*
+
Vuelva a intentar la operación de restauración y cargue las cintas en la secuencia correcta.
