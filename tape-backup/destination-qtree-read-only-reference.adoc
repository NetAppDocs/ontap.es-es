= El qtree de destino es de solo lectura
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensaje*
+
`Destination qtree is read-only`

* *Causa*
+
El qtree al que se intenta restaurar es de solo lectura.

* *Acción Correctiva*
+
Intente restaurar los datos en una ubicación diferente.
