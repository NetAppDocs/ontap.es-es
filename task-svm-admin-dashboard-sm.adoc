= Administra y supervisa los SVM de ONTAP con System Manager
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
A partir de ONTAP 9.19.1, los administradores de SVM pueden supervisar y gestionar sus propias SVM mediante un panel de control dedicado de System Manager.

.Antes de empezar
Tu administrador de clúster debe proporcionarte lo siguiente:

* Dirección IP de gestión al SVM
* Credenciales (nombre de usuario y contraseña) para el panel de control de System Manager de SVM (el usuario vsadmin debe estar desbloqueado)
* Opcionalmente, permisos para añadir volúmenes en la SVM


.Pasos
. Introduce la dirección IP de gestión de la SVM en un navegador.
. Introduce tu nombre de usuario y contraseña para el administrador de SVM (vsadmin).
. Selecciona *Sign in*.
+
Se abre el panel de control de ONTAP System Manager.

. Puedes consultar la siguiente información sobre la SVM:
+
[cols="1,2"]
|===
| Información SVM | Descripción 


| Protocolos | Los protocolos activados en la SVM. 


| Detalles | Información de NIS y LDAP, idioma y el periodo de retención después de que se elimina un volumen. 


| Rendimiento | Estadísticas de latencia, IOPS y rendimiento por hora, día, semana, mes o año para el protocolo seleccionado. 
|===
. Opcionalmente, puedes descargar los datos en un archivo de valores separados por comas (csv) para analizarlos después.
. Seleccione *almacenamiento > volúmenes*.
+
Se abre la página del panel Volumes.

+
El administrador del clúster puede concederte permiso para añadir volúmenes a la SVM. Si no se concede este permiso, al seleccionar el botón *Add* falla y se muestra un mensaje de error.
