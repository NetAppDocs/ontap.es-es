= Requisitos de los puertos de red
:allow-uri-read: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
SnapMirror la replicación de sincronización activa tiene varios requisitos de puerto de red.

Por ejemplo, cada nodo implicado en la replicación debe supervisar el estado de sus nodos asociados, controlar la actividad de replicación y los propios datos deben ser replicados. Además, el ONTAP Mediator o ONTAP Cloud Mediator necesitan comunicarse con todos los nodos para proporcionar servicios de quórum y ayudar con la conmutación por error. Todas estas funciones tienen diferentes requisitos de punto final y puerto.

En la mayoría de los casos, la replicación de sincronización activa de SnapMirror funcionará en una red interna sin restricciones basadas en puertos o destinos. Si el tráfico IP está restringido por cortafuegos, enrutadores, topología de red u otros mecanismos, se deben validar los siguientes requisitos de conectividad.
