= Elimina las SVM activas-activas antes de revertir desde ONTAP 9.19.1
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
La función de multivía activo-activo de SAN no es compatible con ONTAP 9.18.1 y versiones anteriores. Antes de revertir desde ONTAP 9.19.1, debes eliminar cualquier máquina virtual de almacenamiento (SVM) con configuraciones activo-activo.

.Pasos
. Identifica cualquier SVM que tenga configurada la opción san-multipathing para active-active o active-active-scsi-only:
+
[source, cli]
----
vserver show
----
. Elimina cualquier SVM configurada para multivía activo-activo o activo-activo-solo-SCSI:
+
[source, cli]
----
vserver delete -vserver <SVM>
----
. Comprueba que las SVM se hayan eliminado:
+
[source, cli]
----
vserver show
----


.Información relacionada
Más información sobre link:../san-admin/multipath-automatic-lif-failover-support.html["Compatibilidad con multivía activo-activo de SAN"] en ONTAP 9.19.1 y versiones posteriores.
