= Elimina las segundas claves de acceso S3 antes de revertir desde ONTAP 9.19.1
:allow-uri-read: 


[role="lead"]
Si actualizaste a ONTAP 9.19.1 y añadiste una segunda clave de acceso S3 a cualquier usuario, estas claves deben eliminarse antes de que puedas volver a una versión anterior de ONTAP.

.Pasos
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["Mostrar claves de acceso S3"] para determinar si se han añadido segundas claves de acceso para algún usuario S3.
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["Retira las segundas llaves de acceso"] para todos los usuarios de S3.
