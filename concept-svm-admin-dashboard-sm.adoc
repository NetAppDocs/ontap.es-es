= Conoce el panel de control del administrador de ONTAP System Manager SVM
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
A partir de ONTAP 9.19.1, los administradores de SVM pueden supervisar y gestionar sus SVM mediante un panel de control dedicado de System Manager y la página Storage > Volumes. Esto resulta especialmente útil en un entorno ONTAP con inquilinos, donde el administrador del clúster puede proporcionar un panel de control de SVM para cada inquilino.

El administrador del clúster proporciona las credenciales de inicio de sesión a los administradores de SVM. Los administradores de SVM pueden ver los detalles y el rendimiento en el panel de control de SVM y en la página *Storage > Volumes*. El panel de control muestra los protocolos de SVM, las interfaces de red, los detalles y las métricas de rendimiento. La página Volume proporciona información sobre los volúmenes de SVM y, si el administrador del clúster lo permite, la posibilidad de que el administrador de SVM añada nuevos volúmenes.
