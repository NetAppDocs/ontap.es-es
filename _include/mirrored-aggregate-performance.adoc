NOTE: Se recomienda mantener al menos un 20 % de espacio libre para agregados reflejados para lograr un rendimiento y una disponibilidad de almacenamiento óptimos.  Aunque la recomendación es del 10% para agregados no reflejados, el sistema de archivos puede usar el 10% adicional de espacio para absorber cambios incrementales.  Los cambios incrementales incrementan la utilización del espacio para los agregados reflejados debido a la arquitectura basada en instantáneas de redirección en escritura de ONTAP.  El incumplimiento de estas prácticas recomendadas puede tener un impacto negativo en el rendimiento.
