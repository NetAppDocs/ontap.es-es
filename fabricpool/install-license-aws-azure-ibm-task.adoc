= Instale una licencia de FabricPool en un clúster de ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Cloud Tiering dejará de estar disponible para la compra o renovación de licencias a partir del 26 de abril de 2026. Si estás utilizando NetApp Cloud Tiering, NetApp te recomienda que trabajes con tu representante de NetApp para hacer la transición de tus licencias de tiering existentes a licencias de ONTAP FabricPool.

Los clientes existentes pueden seguir utilizando y recibiendo asistencia para NetApp Cloud Tiering hasta que caduque su contrato de suscripción o licencia. Después de que la suscripción caduque, ya no tendrás acceso a las funciones ni al soporte de NetApp Cloud Tiering.

Si actualmente estás usando licencias de Cloud Tiering, no te verás afectado hasta que tu licencia de Cloud Tiering caduque o alcance su capacidad máxima. Contacta a NetApp cuando necesites actualizar tu licencia o antes para asegurarte de que no haya interrupciones en tu capacidad de trasladar datos a la nube.

link:https://docs.netapp.com/us-en/data-services-cloud-tiering/task-licensing-cloud-tiering.html["Más información sobre los cambios de licencia de NetApp Cloud Tiering"^].

Puedes descargar y activar una licencia de FabricPool usando System Manager.

La licencia FabricPool es una licencia para todo el clúster. Incluye un límite de uso autorizado que ha adquirido para el almacenamiento de objetos asociado con FabricPool en el clúster. El uso en todo el clúster no debe superar la capacidad del límite de uso autorizado. Si necesita aumentar el límite de uso de la licencia, debe ponerse en contacto con su representante de ventas.

Las licencias de FabricPool están disponibles en formatos perpetuas o basados en períodos de 1 o 3 años.

Una licencia FabricPool basada en plazos con 10 TB de capacidad gratuita está disponible para los primeros pedidos FabricPool para configuraciones de clústeres existentes. La capacidad gratuita no está disponible con licencias perpetuas. No se requiere una licencia si usas NetApp StorageGRID u ONTAP S3 para el nivel de nube. Cloud Volumes ONTAP no requiere una licencia FabricPool, sin importar el proveedor que estés usando.

.Antes de empezar
Necesitas el UUID de tu clúster para obtener tu archivo de licencia de NetApp (NLF) desde el sitio de soporte de NetApp.

.Pasos
Esta tarea solo se admite cargando el archivo de licencia al clúster mediante System Manager.

. Descargue el archivo de licencia de NetApp (NLF) para la licencia de FabricPool en el link:https://mysupport.netapp.com/site/global/dashboard["Sitio de soporte de NetApp"^].
. Realice las siguientes acciones mediante System Manager para cargar la licencia de FabricPool en el clúster:
+
.. En el panel *Cluster > Settings*, en la tarjeta *Licenses*, haga clic en image:icon_arrow.gif["Icono de flecha"].
.. En la página *License*, haga clic en image:icon_add.gif["Icono Agregar"].
.. En el cuadro de diálogo *Agregar licencia*, haga clic en *examinar* para seleccionar el NLF que descargó y, a continuación, haga clic en *Agregar* para cargar el archivo en el clúster.
