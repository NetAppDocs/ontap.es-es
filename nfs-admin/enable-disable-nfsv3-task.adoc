= Habilite o deshabilite NFSv3
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puede habilitar o deshabilitar NFSv3 modificando el `-v3` opci칩n. De este modo, los clientes pueden acceder a archivos mediante el protocolo NFSv3. De forma predeterminada, NFSv3 est치 habilitado.

.Paso
. Ejecute una de las siguientes acciones:
+
[cols="20,80"]
|===


| Si desea... | Introduzca el comando... 


 a| 
Habilite NFSv3
 a| 
`vserver nfs modify -vserver vserver_name -v3 enabled`



 a| 
Desactive NFSv3
 a| 
`vserver nfs modify -vserver vserver_name -v3 disabled`

|===
